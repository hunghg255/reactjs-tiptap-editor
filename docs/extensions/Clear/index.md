# Clear

The Clear extension allows you to clear the editor content.

## Usage

```tsx
import { Clear } from 'reactjs-tiptap-editor/clear'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  Clear // [!code ++]
];
```
