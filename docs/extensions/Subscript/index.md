# Subscript

The Subscript extension allows you to subscript text in your editor.

## Usage

```tsx
import { Subscript } from 'reactjs-tiptap-editor'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  Subscript // [!code ++]
];
```
